Mac OSX Vim 输入中文乱码问题

出现问题的环境:

  1. OSX 10.9.2 英文
  2. VIM 7.3

问题描述:

  1. 可以正常显示文件中得中文字符
  2. 输入中文字符会出现乱码
  3. 通过在.vimrc中设置编码格式无效

解决方案:

找到 Terminal-->Preferences-->Settings-->Advanced-->Escape non-ASCII input with Control-V 取消该选项即可。
