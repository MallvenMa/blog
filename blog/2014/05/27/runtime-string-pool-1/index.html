JVM 之 String 常量池 一

1.Sting对象创建方式

JAVA里面创建字符串有两种方式：
1. 通过字符串常量方式:String a = “a”;
2. 通过new 关键字创建:new String(“a”);

但是这两种创建字符串的方式有很大的不同。
1. 使用第一种方式，jvm首先会检查字符串常量池中是否存在了这个常量，如果存在，就返回池中的实例引用。如果不存在，就会实例化一个字符串并放到池中,然后返回引用。
2. 使用第二种方式，则直接分配到heap中，不指向字符串常量池中的任何对象，和字符串常量池没有关系。
